OnePass docs, built with Next.js and MDX.

## How to install

1. Clone this repository:

```bash
git clone https://github.com/Cores-ts/onepass.docs.git
```

1. Go into project

```bash
cd onepass.docs
```

1. Install dependencies

```bash
yarn
```

1. Run project

```bash
yarn dev
```
